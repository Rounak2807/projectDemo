Step to running the project.

1.First run the sql script in oracle db.
2.import the project as maven project.
3.Force update the maven project 
   3(a).Right click on the project 
   3(b).Select maven update and do force update so that it will download all the related jar file.
4.In Spring-servelt.xml file change the username and password and the connection url for which we want to run the application.
5.After that run the application and do the requried operation.
